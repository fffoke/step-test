# Step Test Project

Групповой проект Django для изучения работы с Git и ветками.

## Описание проекта
Это Django проект с двумя приложениями:
- **users** - управление пользователями (регистрация, вход)
- **shop** - магазин с товарами

## Участники группы
- Участник 1
- Участник 2  
- Участник 3

## Установка и запуск

1. Установите зависимости:
```bash
pip install django pillow
```

2. Выполните миграции:
```bash
python manage.py migrate
```

3. Запустите сервер:
```bash
python manage.py runserver
```

## Структура проекта
```
config/ - основные настройки Django
shop/ - приложение магазина
users/ - приложение пользователей
media/ - загруженные файлы
```

## Работа с ветками
Каждый участник создает свою ветку для внесения изменений.
